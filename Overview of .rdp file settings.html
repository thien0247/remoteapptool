Overview of .rdp file settings

Home/Overview of .rdp file settings
Overview of .rdp file settingsDonkz2022-12-11T21:51:39+01:00

On this page you will find an overview of most of the available .rdp file settings which can be used with the /o command line switch.

All settings must be specified using the .rdp file style syntax:

option:type:value

Examples:
alternate shell:s:notepad.exe
keyboardhook:i:2

Note: The information in this overview is largely compiled from this article at the Microsoft TechNet site.

Use the search box on the left to search and filter keywords.
